Interactive Red Genetically Encoded Calcium Indicator Benchmark Test

This site is intended as a reference for both experimentalists and computational biologists interested in the design or use diversity of red genetically encoded calcium indicators (GECIs). This interactive dashboard provides an overview of the experimental benchmarking results for a series of GECIs from Berndt Lab, UW. Each interactive panel below corresponds to a distinct stage of characterization — from in vitro titration curves of purified proteins, to cellular time traces recorded under field stimulation in primary cultured rat cortical neurons, and summary plots that highlight key functional parameters. You can use the legends to dynamically select or deselect individual GECIs for direct side-by-side comparison. Hover over any curve or data point to reveal detailed numerical values, fitted parameters, and confidence intervals. All panels are fully interactive — supporting zoom, pan, and direct export of figures from the interface. This dashboard is an ongoing project, and future updates may include additional variants, experimental conditions, and expanded benchmarking datasets. If you have any questions, suggestions, or encounter any display issues, please feel free to contact lishuyu@uw.edu.
